Сервис API реализованный на Flask

Описание основных функций:

Регистрация пользователя

[POST] http://127.0.0.1:5051/user/register/?name=[PARAMS]&email=[PARAMS]&pwd=[PARAMS]

[PARAMS] - заменить своими значениями

Вход в систему

[POST] http://127.0.0.1:5051/user/login/?name=[PARAMS]&pwd=[PARAMS]

[PARAMS] - заменить своими значениями

Выход из системы

[POST] http://127.0.0.1:5051/user/logout

Добавление объявление

[POST] http://127.0.0.1:5051/advert/?title=[PARAMS]&description=[PARAMS]

[PARAMS] - заменить своими значениями

Получить объявление

[GET] http://127.0.0.1:5051/advert/[PARAMS]

[PARAMS] - ID номер объявления

Редактировать объявление

[PATCH] http://127.0.0.1:5051/advert/[PARAMS]/?description=[NEW PARAMS]

[PARAMS] - ID номер объявления
[NEW PARAMS] - Новое значение

Удалить объявление

[DELETE] http://127.0.0.1:5051/advert/[PARAMS]

[PARAMS] - ID номер объявления
